## What this is
An interactive, AI-powered strategy deck built for the eos Products Manager of GenAI Strategy & Transformation interview. Not a slide deck — a working prototype.

## What's inside
- **7 interactive slides** — fully navigable by keyboard, click, or tap
- **Live Social Pulse Engine** — real Claude API call analyzing eos community signals
- **Aria** — eos AI Enablement Agent powered by Claude
- **Prompt Library** — links to live portfolio
- **eos brand design** — matches evolutionofsmooth.com palette

## Tech stack
- Vanilla HTML/CSS/JS — no framework, no build step needed
- Claude API (Anthropic) — powers Social Pulse demo + Aria
- Vercel — static deployment

## How to run locally
Just open `index.html` in any browser. No server needed.

## How to deploy to Vercel
1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import your repo
3. Deploy — done. Your live URL appears instantly.
